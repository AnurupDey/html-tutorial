LISTS

We often find it necessary to display a list of things. Those familiar with word will remember the bullets and numbered lists options. HTML also allows us create such lists. In fact it allows us to create three kinds of lists, they are:-

  1. Ordered lists
  2. Unorderd list
  3. Description list

ORDERED LISTS

These lists are very similar to numbered lists created in MS word. In this case the item is proceeded by an Arabic numeral(1,2,3,… DEFAULT), upper or lower case Roman numerals(I,ii,iii,… or I,II,III,…), or by upper or lower case alphabet tags(a,b,c… or A,B,C,… ).

SYNTAX:

These lists are enclosed between the <OL> and </OL> tags. Further, each item of the list is enclosed within <LI> and </LI> tags.

EXAMPLE:

<OL> <LI> Apples</LI> <LI> Oranges</LI> <LI> Grapes</LI> </OL> The display will be:

  1. Apples
  2. Oranges
  3. Grapes

HTML also allows you to specify how we want to number our lists. This can be done by using the TYPE attribute with the <OL> tag. Listed below are some of the values and their effects when used with the TYPE attribute.

TYPE OUTPUT
TYPE = “1” 1,2,3,…
TYPE = “A” A,B,C,…
TYPE = “a” a,b,c,…
TYPE = “I” I,II,III,…
TYPE = “i” i,ii,iii,…

We can use START attribute to specify the starting number of the list.


EXAMPLE: <OL TYPE=“ 1” START=“ 4”> will display the list as 4,5,6,…

UNORDERED LIST

These are like the bullet points created in MS word. In this case, a round or square bullet proceeds the list items.

SYNTAX:
These lists are enclosed within <UL> and </UL> tags. Each item of the list is enclosed within <LI> and </LI> tags.

EXAMPLE:

<UL> <LI>Potato</LI> <LI>Radish</LI> <LI>Onion</LI> </UL> The display will be:

  • Potato
  • Radish
  • Onion


The TYPE attribute can be used with the <UL> tag to specify the bullet style.

TYPE Result
TYPE = “Disc”
TYPE = “Square”
TYPE = “Circle”
TYPE = “None”

DESCRIPTION LIST

As the name indicates, these lists are used for describing terms.


SYNTAX:
They are enclosed within the <DL> and </DL> tags. Each definition term is enclosed within the definition term tags, <DT> and </DT>. Each definition is enclosed within the actual definition tag,<DD> and </DD>.

EXAMPLE:

<DL> <DT> Dir</DT> <DD> MS-DOS command used to display the contents of the disk.</DD> <DT> Copy</DT> <DD> MS-DOS command used to make duplicate copy of files.</DD> <DT> Del</DT> <DD> MS-DOS command used to delete files.</DD> </DL> The display will be:
Dir
MS-DOS command used to display the contents of the disk.
Copy
MS-DOS command used to make duplicate copy of files.
Del
MS-DOS command used to delete files.
