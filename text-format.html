HTML FORMATTING TAGS

HTML Formatting Elements HTML defines special elements for defining text with a special meaning. HTML uses elements like <b> and <i> for formatting output, like bold or italic text. Formatting elements were designed to display special types of text. some of them are as follows:

<b> AND <i> TAG - BOLD AND ITALIC TEXT

Ex:

<!DOCTYPE html> <html> <body> <p> This is normal text <b>This text is bold</b> <br> <i>This text is italic</i> <br> </p> </body> </html>

Display:

This text is normal.
This text is bold
This text is italic


<strong> TAG - DEFINES IMPORTANT TEXT

It is a phrase tag.

Ex:

<!DOCTYPE html> <html> <body> <p> This text is normal. <br> <strong>This text is important.</strong> </body> </html>
Display:
This text is normal.
This text is important.

<em> TAG - RENDERS AS EMPHASIZED TEXT

Ex:

<!DOCTYPE html> <html> <body> <p>This text is normal.</p> <p> <em>This text is emphasized.</em> </p> </body> </html>
Display:

This text is normal.

This text is emphasized.

<small> TAG - DEFINES SMALLER TEXT

Ex:

<!DOCTYPE html> <html> <body> HTML <small>Small</small> Formatting. </body> </html>
Display:

HTML Small Formatting

<mark> TAG - DEFINES HIGHLIGHTED TEXT

Ex:

<!DOCTYPE html> <html> <body> <h2>HTML <mark>Marked</mark> Formatting</h2> </body> </html>
Display:
HTML Marked Formatting.

<del> TAG - DEFINES DELETED (REMOVED) TEXT

ex:

<!DOCTYPE html> <html> <body> This text is normal. <br> <del>This text is deleted.</del> </body> </html>
Display:
This text is normal.
This text is deleted.

<ins> TAG - DEFINES INSERTED (ADDED) TEXT

Ex:

<!DOCTYPE html> <html> <body> <p>My favorite <ins>color</ins> is red.</p> </body> </html>
Display:

My favorite color is red.


<sub> AND <sup> TAG - SUBSCRIPT AND SUPERSCRIPT TEXT

Ex:

<!DOCTYPE html> <html> <body> <p> This is normal text. <br> This is <sub> subscript</sub> and <sup>superscript</sup> </p> </body> </html>
Display:

This text is normal.
This is subscript and superscript.
